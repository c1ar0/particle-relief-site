# 光点粒子减压小游戏

无需构建的静态网页。页面是全屏 Canvas 体验：低刺激配色、手机端锁定视口不滚动，隐藏设置默认不打扰。

## 已补齐功能

1. 声音反馈：首次交互后启用 Web Audio 柔和提示音，可在隐藏设置关闭。
2. 震动反馈：支持 `navigator.vibrate` 的设备会在点击、长按、双击、捏合时轻微震动。
3. 设备方向：支持 DeviceOrientation，倾斜手机会轻微改变光团与粒子漂移方向。
4. 多主题：深夜、晨雾、林息、余温四套低刺激主题。
5. 隐藏设置：右上角隐形热区点击 / 长按，或键盘 `S` / `?` 打开，可切换声音、震动、方向、无压力目标与主题。
6. 双击：双击或快速两次轻触切换正常 / 更慢节奏。
7. 捏合旋转：双指捏合会扩散 / 聚拢粒子，双指旋转会产生旋涡场。
8. 无压力目标：画面中有淡淡的呼吸圆环，靠近或停留会自然完成并生成舒缓反馈；可关闭。
9. 多指 / 多指针互动：每个触点都会影响粒子。
10. 适配 `prefers-reduced-motion`，降低粒子数量、速度、声音与震动刺激。

## 玩法

- 点击 / 轻触：生成一圈光点涟漪
- 按住拖动：持续释放粒子轨迹
- 长按：触发更舒缓的聚拢效果
- 双击：切换更慢 / 正常节奏
- 双指捏合：聚拢或释放粒子
- 双指旋转：制造轻柔旋涡
- 倾斜手机：让粒子缓慢偏移
- 右上角隐藏热区 / `S` / `?`：打开隐藏设置

## 运行

直接打开 `index.html`，或在目录中启动静态服务：

```bash
python3 -m http.server 8000
```

访问：http://localhost:8000
